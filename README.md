# 蜜袋鼯粮店微信小程序

这是一个可连接真实后端的微信小程序样例，包含：

- 商品浏览
- 购物车
- 地址管理
- 微信地址导入
- 下单前支付前置
- 订单状态细化
- 管理员订单列表与人工改状态

## 目录

- `pages/`：小程序页面
- `utils/`：本地数据层、接口封装、会话管理
- `backend/`：本地 Node 后端和管理员页面

## 启动后端

```powershell
node .\backend\server.js
```

默认地址：

- 接口：`http://127.0.0.1:3000`
- 管理页：`http://127.0.0.1:3000/admin`
- 默认管理员令牌：`changeme`

可通过环境变量修改：

- `PORT`
- `ADMIN_TOKEN`
- `WX_APPID`
- `WX_APPSECRET`

## 小程序接入

在微信开发者工具里，把小程序接口地址指向你的后端。

本地开发时可以直接使用 `http://127.0.0.1:3000`。
正式环境请使用 HTTPS 域名，并在小程序后台配置合法域名。

## 订单流程

1. 选择商品，进入确认订单页
2. 选择或导入收货地址
3. 提交订单后进入 `pending_payment`
4. 在订单详情页完成支付，状态变为 `paid`
5. 管理员在后台把订单推进到 `preparing`、`shipping`、`done`

## 地址管理

支持两种地址来源：

- 微信地址导入
- 手动新增、编辑、设默认

地址数据会同步到订单快照，避免后续修改地址影响历史订单。

## 管理员页面

管理员页面提供：

- 订单列表
- 状态筛选
- 快速切换状态
- 订单详情查看
- 物流号和物流说明维护

如果后续要接入真正的后台管理系统，可以把 `/admin` 这一层替换成独立前端，再复用同一组 API。
